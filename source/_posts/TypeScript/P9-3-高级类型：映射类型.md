## 映射类型

> 可以通过一个旧的类型生成一个新的类型

举个例子：

``` typescript
interface Obj {
  a: number;
  b: string;
  c: boolean
}

type ReadonlyObj = Readonly<Obj>
// 会输出：
// {
// readonly a: number;
// readonly b: string;
// readonly c: boolean
// }
```

利用 `ts` 中的内置类型别名，能输出一些映射类型，而映射类型又分为：

1. 同态类型
2. 非同态类型

上面的 `Readonly` 输出的是同态类型。

再举个例子： 输出一个非同态类型

``` typescript
type RecordObj = Record<'x' | 'y', Obj>
// 会输出：
// {
// x: Obj;
// y: Obj
// }
```
