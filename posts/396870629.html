TypeScript 较 JavaScript

TypeScript 是什么

TypeScript （简称 TS） 是 JavaScript 的 超集 （JS 有的 TS 也有）

TypeScript = Type + JavaScript （为 JS 添加类型系统）

TS 相比 JS 的优势

  • 优势一：类型化思维方式，使得开发更加严谨，提前发现错误，
  • 优势二：类型系统提高了代码可读性，并使维护和重构代码更加容易
  • 优势三：补充了接口、枚举等开发大型应用时 JS 缺失的功能
